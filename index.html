QPE Demo

Synthetic data only — no real plans or projects

QPE is a deterministic quarterly planning engine. Given a scope of work and team capacity, it produces an allocated plan, scores scenario changes against that baseline, and generates read-model reports for review.

This public demo uses synthetic data only. It shows how a team plan, a scenario change, a plan-health view, an actuals variance, and a next-cycle advisory can be reviewed through static read-model reports. Every name, date, figure, and identifier is fabricated and does not represent any real organisation, project, or individual.

The demo is read-only. No user data is uploaded or stored.
